# 目鏡大叔部落格文章排版規範

> 最後更新：2026-01-30
> 用途：確保所有文章風格一致，方便 AI 助理產出符合格式的初稿

---

## 文章結構總覽

```
1. Meta 標籤區（SEO）
2. 內嵌 CSS（固定樣式）
3. 作者署名
4. 首圖 + 圖說
5. 開場白（場景/情境切入）
6. H2 分段內容（2-5 個段落）
7. H3 子段落（視需要）
8. 重點提示框（blockquote.note）
9. Hashtag 標籤列
10. 大叔的承諾（blockquote.note）
11. 作者名片 + CTA 卡片（store-card）
12. FAQ 區塊
13. Schema 結構化資料（Article + FAQPage）
```

---

## 寫作風格提醒

1. **開場**：用場景/畫面開頭，不要「今天要來談...」
2. **對話**：穿插真實對白，增加臨場感
3. **語氣**：口語化、有台語感（「欸不是」「啊就」）
4. **專業**：用比喻解釋，不堆術語
5. **結尾**：金句收尾 + 溫暖提問
6. **法律**：參考《驗光師文案法律須知.md》
