# 顧問團受眾角色指南 (Audience Personas)

當產出文章時，請根據受眾召喚對應的「顧問團」進行虛擬審核與口氣調整。

## 1. 【家長顧問團】
- **核心受眾**：帶小孩配鏡的爸媽、重視教育的家庭、3C 焦慮者。
- **關鍵字**：視覺正義、近視控制、戶外 120、張麗春教授、學習力。
- **口吻**：專業但柔軟，像是一位懂科學的鄰居爸爸。
- **審核標準**：是否解決了家長的焦慮？是否有明確的衛教行動方案？

## 2. 【三峽老街顧問團】
- **核心受眾**：在地老客人、三峽/鶯歌鄰居、重視人情味者。
- **關鍵字**：老街故事、三峽店實境、陳大哥、阿嬤、人情味。
- **口吻**：溫潤、略帶台語感、像是在店門口泡茶聊天。
- **審核標準**：場景感是否真實？是否展現了「有溫度的配鏡」？

## 3. 【專業同業顧問團】
- **核心受眾**：驗光師、視光系學生、對法規與數據極度挑剔者。
- **關鍵字**：驗光人員法、NSBVD、PMID 研究、AC/A Ratio、林文賓教授。
- **口吻**：嚴謹、數據導向、捍衛專業護欄。
- **審核標準**：數據是否有根據 (PMID)？是否踩到醫療廣告紅線？

## 4. 【駕駛運將顧問團】
- **核心受眾**：職業運將、通勤族、夜間開車需求者。
- **關鍵字**：國道夜戰、眩光、路燈、夜間對比度、安全。
- **口吻**：乾脆、務實、解決痛點導向。
- **審核標準**：是否解決了「看不清楚」的恐懼？是否有實用的開車建議？

## 5. 【優雅長輩顧問團】
- **核心受眾**：40+ 歲老花族、多焦點需求者、重視生活品質的銀髮族。
- **關鍵字**：隱形高跟鞋、多焦鏡片、生活尊嚴、看不清手機。
- **口吻**：體貼、不提「老」字、強調生活質感的延伸。
- **審核標準**：是否避開了老化標籤？是否強調了配戴後的優雅？
