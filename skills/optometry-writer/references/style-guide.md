# 驗光師。目鏡大叔 部落格排版與規範

## 1. 基礎排版符號
- **H2 標題**：使用「▮」符號。例：`## ▮ 標題名稱`
- **H3 標題**：使用「▸」符號。例：`### ▸ 次標題名稱`
- **內文**：字級較大，行高較寬（1.8）。

## 2. 特殊區塊 (CSS Class)
- **重點提示框**：使用 `blockquote.note`。
- **作者名片**：使用 `div.store-card`。
- **引用**：場景開場或對話穿插。

## 3. SEO 與 Schema
- 每篇必含 `Article` Schema。
- 若有問答，必含 `FAQPage` Schema。
- Meta Description 需包含地區關鍵字（三峽、鶯歌、樹林、土城）。

## 4. 文風指南
- **場景優先**：不從理論開始，從一個故事或一個客人的問題開始。
- **口語穿插**：適度加入台語感或自嘲幽默。
- **金句收尾**：文章最後要有一段具溫度的總結。

## 5. 法律紅線與 EEAT 強化 (YMYL 規範)
- **Experience (經驗)**：每篇長文必須包含至少一個「大叔在三峽店內」的真實觀察或案例。
- **Expertise (專業)**：文末統一標註作者資歷（驗光師、學會理事、元培講師）。
- **Authoritativeness (權威)**：若涉及近視控制、雙眼視覺，必須引用內部文獻 (PMID) 或張麗春教授等專家觀點。
- **Trustworthiness (信任)**：
    - 加入「醫事建議聲明」：本文僅供衛教參考，不能取代專業面診。
    - 語氣中立：避免使用「絕對」、「保證」等誇大詞彙。
    - 提供實體門市資訊（三峽店）。
