# AI Prompts & Strategic Toolbox

## General Knowledge Tools
- **Feynman Breakdown (快速學習)**: 簡單解釋、抓盲點、重整邏輯、生活類比。
- **Chapter Mastery Guide (章節指南)**: 核心總結、學習目標、結構大綱、戰略公式、魔王關卡。
- **Research Brief (權威簡報)**: 戰情背景、研究工法、關鍵論點、行動建議。
- **Decision Matrix (決策矩陣)**: 決策指標、權重分配、損益風險、信心指數。
- **High-Signal Extraction (情資提煉)**: 脫水處理、決策優先、戰略價值說明。
- **Contradiction Finder (衝突偵測)**: 來源對槓、看穿假設、調解方案、補完計畫。
- **Claim-Evidence Map (論證地圖)**: 核心論點、支持證據、來源定位、強度評估、假設、反面證據。

## Content Creation Tools
- **Multi-Format Content Architecture (多通路架構)**: 部落格大綱、YouTube 腳本、X 推文連貼。
- **Infographic Architecture (視覺藍圖)**: 視覺層級、拒絕冗餘、文圖雙重強化。
- **Pattern Break Architect (反直覺內容)**: 打破平庸、創造意外感、視覺/Hook 建議。
- **Attention-Hijacking Hook Architect (流量抓手)**: 5 個止滑開場白、心理分析。
- **Silent Content Multiplier (不露臉工廠)**: 文字/圖表 Carousel、極大化收藏。
- **Algorithm-Friendly Rewrite (演算法優化)**: 結構導引、節奏控管、極大化完讀率。
- **Scroll Retention Engineer (完讀率工程)**: 逐行導引、視覺留白、手機閱讀優化。

## Learning & Memory Tools
- **Active Recall Challenge (練功房)**: 基礎題、應用題、魔王題、詳解。
- **Strategic Patrol & Synthesis (戰略巡航)**: 情資偵查、跨界聯覺、資產建議、預發大綱。
