## 🔬 MC-001: Ortho-K Meta Analysis (2026)

### 基本資料
- **標題**：Orthokeratology for Myopia Control: A Systematic Review and Meta-Analysis
- **年份**：2026
- **來源**：PMC12917334
- **研究類型**：系統回顧與統合分析

### 研究對象
- **人數**：1,065 名兒童/青少年
- **年齡**：6-18 歲
- **樣本**：15 項隨機及前瞻性對照試驗

### 主要結果
| 指標 | 結果 |
|------|------|
| 軸長增長減少 | **0.15 mm** (12個月) |
| 95% 信賴區間 | -0.20 至 -0.10 mm |
| I² 異質性 | 90.8% |

### 機制發現
- **周邊近視離焦**：增強周邊離焦設計可能提供額外益處
- **效果持續性**：效果持續至 3 年

### 限制
- 多為東亞樣本
- 需要更多多民族長期試驗

### 🔑 重點引用
> 「12±2個月內，Ortho-K組軸長增長減少0.15 mm」

### 臨床意義
- Ortho-K 是有效的近視控制方法
- 設計優化（增強周邊離焦）可提升效果
- 需要個別化評估

## 📋 MC-003: 葉黃素兒童近視輔助 (2026)

### 基本資料
- **標題**：Lutein as Myopia Management Supplement
- **年份**：2026
- **來源**：中國雙盲 RCT
- **PMIDs**：待查

### 研究對象
- **人數**：180 名兒童
- **年齡**：8-12 歲
- **時間**：6 個月

### 劑量
- **每日劑量**：8 mg 葉黃素

### 主要結果
| 指標 | 變化 |
|------|------|
| 中央脈絡膜厚度 | 抑制下降 10-15 µm |
| 周邊脈絡膜厚度 | 抑制下降 10-15 µm |
| 等效球度 (SER) | 無顯著變化 |
| 軸長 (AL) | 無顯著變化 |

### 結論
- **安全且經濟**的輔助方案
- 無 SER/AL 顯著變化
- 效果類似 1 個月 Ortho-K

### ⚠️ 重要提醒
- 葉黃素**無法直接預防近視**
- 可作為**輔助治療**
- 核心仍是「戶外活動 + 控制近距離用眼」

## ✅ MC-004: LAMP 阿托品研究 (2019)

### 基本資料
- **標題**：Low-Concentration Atropine for Myopia Progression (LAMP)
- **年份**：2019
- **PMID**：30514630
- **期刊**：Ophthalmology
- **機構**：香港中文大學 (CUHK)

### 研究設計
- 隨機、雙盲、安慰劑對照試驗
- 4-12 歲兒童
- 評估 0.05%, 0.025%, 0.01% 三種濃度

### 主要結果
| 濃度 | 2年控制率 |
|------|-----------|
| 0.01% | ~27% |
| 0.025% | ~43% |
| 0.05% | **~50-60%** (最優) |

### 副作用
- 劑量依賴：濃度越高，瞳孔放大越多
- 畏光、調節力下降（停藥可恢復）
