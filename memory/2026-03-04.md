## 重要發現與決策紀錄

### 1. Article Schema 升級版
比 `references/部落格排版規範.md` 中的舊版 Schema 更完整，新增了：
- `author.affiliation`：4 個組織（自己的眼鏡、元培、學會、公會）
- `author.knowsAbout`：專長標籤（兒童近視控制、驗光配鏡、視力保健、角膜塑型片）
- `author.url`：About 頁面連結
- `keywords`：文章關鍵字陣列
- `datePublished` / `dateModified`

**建議**：以後所有文章的 Article Schema 都用這個升級版。範本在 `遮雨棚下的一堂課-完整版.html` 頂部可查。

### 2. SEO 標題策略
經分析，標題用 **【三峽驗光師】** 比 **【新北市驗光師】** 更有效：
- 「三峽」搜尋意圖精準、競爭者少、Local SEO 加分
- 「新北市」在內文地址自然出現即可，不需放標題
- 內文自然植入 1-2 次「三峽兒童近視控制」「三峽驗光所」

### 3. 數據來源升級
- **舊**：國健署 106 年「兒童青少年視力監測調查」（小一 19.8%→小六 70.6%）
- **新**：教育部 93～113 學年度學生視力檢查統計（小一不到 3 成→小六超過 6 成→國中 7 成以上）
- 新數據來源更權威、更新、跨度更長（20 年）
- 內部連結到大叔自己的分析文章：https://www.uncle-glasses.net/2025/08/taiwan-student-vision-report-93-113.html

### 4. 三種平台的語言差異

| 平台 | 核心策略 | 語言風格 |
|------|----------|----------|
| **Blogger** | SEO + Schema + 數據 + 完整專業度 | 完整敘事、有數據段、有 FAQ |
| **Facebook** | 故事驅動、鏡頭感、情緒共鳴 | 短句斷行、身體語言描寫、留白製造節奏、不放數據、連結放留言區 |
| **LINE** | 快速轉傳、精華摘要 | emoji 分隔、核心故事 + 3 件事 + CTA |

### 5. FB 受眾偏好
目鏡大叔的 FB 受眾喜歡：
- **故事**（越有畫面感越好）
- **鏡頭感**（像在看短片）
- 短句換行、emoji 當視覺錨點
- Hook 放前 3 行（FB「⋯⋯更多」之前）
- 連結放留言區第一則（避免觸及率下降）
- Hashtag：#三峽驗光師 #兒童近視控制 #近視控制 #目鏡大叔 #自己的眼鏡

### 6. E-E-A-T 檢查清單
每篇文章發布前須確認：
- [ ] Experience：基於真實第一手經驗，有具體細節
- [ ] Expertise：引用具體數據（含年份、來源），FAQ 展開專業方法名稱
- [ ] Authoritativeness：作者介紹含三大頭銜，Article Schema 完整
- [ ] Trustworthiness：資料來源標註，語氣「不嚇人但讓你醒」，無推銷感
- [ ] SEO 關鍵字自然植入
- [ ] Blogger「搜尋說明」已提供
- [ ] 雙 Schema（Article + FAQPage）已加入

### 7. 固定元素提醒
每篇文章底部必須有：
- 作者介紹段（含元培講師 + 學會常務理事 + 公會常務理事）
- CTA（地址 + 電話 + LINE 連結）
- Google Maps iframe
- FAQ（4-6 題）
- FAQPage Schema
