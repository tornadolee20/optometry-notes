# 🎯 曼陀羅思考法 - 核心主題展開

## 使用方式

```
選一個核心主題 → 展開成 8 個子議題 → 每個子議題產出一篇文章
```

---

## 曼陀羅九宮格

```
        ┌─────────┬─────────┬─────────┐
        │  子議題  │  子議題  │  子議題  │
        │    1    │    2    │    3    │
        ├─────────┼─────────┼─────────┤
  核    │  子議題  │ 核心主題 │  子議題  │
  心    │    4    │  (Center)│    5    │
  主    ├─────────┼─────────┼─────────┤
  題    │  子議題  │  子議題  │  子議題  │
        │    6    │    7    │    8    │
        └─────────┴─────────┴─────────┘
```

---

## 📝 範例：核心主題「近視控制」

### 中心：近視控制

### 子議題展開（8方向）

| 方向 | 子議題 | 文章標題方向 |
|------|--------|--------------|
| **1. 原因** | 為什麼近視一直增加？ | 「醫師解析：孩子近視加深的 5 個原因」 |
| **2. 方法** | 有哪些控制方法？ | 「2026 最完整：兒童近視控制 3 大方法比較」 |
| **3. 年齡** | 幾歲開始要控制？ | 「几岁该开始近视控制？眼科医师这样说」 |
| **4. 價格** | 一年要花多少錢？ | 「近視控制花費分析：一年到底要多少钱」 |
| **5. 效果** | 哪個方法最有效？ | 「数据说话：阿托品、OK镜、DIMS 效果PK」 |
| **6. 選擇** | 怎麼選適合的方案？ | 「医师 3 问：帮你选对近视控制方案」 |
| **7. 副作用** | 有沒有風險？ | 「父母必看：近视控制的副作用与风险」 |
| **8. 維持** | 控制後要注意什麼？ | 「近视控制不是一劳永逸：后续保养重点」 |

## 🔄 產出流程

```
核心主題
    ↓
┌──────────────────────────────────┐
│  曼陀羅展開成 8 子議題           │
└──────────────────────────────────┘
    ↓
┌──────────────────────────────────┐
│  蜂群產出：每子議題 → 1 篇文章    │
│  Researcher → Writer → Editor   │
└──────────────────────────────────┘
    ↓
┌──────────────────────────────────┐
│  家長會審核：每篇評分            │
└──────────────────────────────────┘
    ↓
┌──────────────────────────────────┐
│  系列文章完成 → 發布              │
└──────────────────────────────────┘
```
