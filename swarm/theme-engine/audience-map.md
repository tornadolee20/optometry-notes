# 🎯 主題永動機 - 族群痛點地圖

## 六大目標族群

| 族群 | 核心痛點 | 關注議題 | 內容需求 |
|------|----------|----------|----------|
| **👶 幼兒家長 (3-6歲)** | 這麼小就近視怎麼辦？ | 假性近視、篩檢、預防 | 安慰+知識 |
| **👦 學齡兒童家長 (6-12歲)** | 度數一直增加 | 近視控制、OK鏡、阿托品 | 方案比較 |
| **👧 青少年 (13-18歲)** | 不想戴眼鏡 | 隱形眼鏡、激光手術 | 美觀/運動 |
| **👨 上班族** | 3C眼疲勞/乾眼 | 電腦眼、乾眼症 | 舒緩方法 |
| **👴 銀髮族** | 看不清楚/白內障 | 老花、白內障手術 | 治療選項 |
| **👓 隱形眼鏡族** | 紅血絲/缺氧 | 角膜健康、舒適度 | 正確清潔 |
