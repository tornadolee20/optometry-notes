## 二、標準配鏡流程（專業版）

### 完整流程（約 30-60 分鐘）

```
1. 問診（5-10 分鐘）
   ├─ 了解用眼需求（工作、生活習慣）
   ├─ 過去配鏡經驗
   ├─ 現有眼鏡問題
   └─ 特殊需求（夜間駕駛、電腦使用等）

2. 驗光檢查（15-20 分鐘）
   ├─ 裸眼視力測量
   ├─ 電腦驗光（初步度數）
   ├─ 綜合驗光儀精測
   │   ├─ 近視/遠視度數
   │   ├─ 散光度數與軸度
   │   └─ 雙眼平衡
   ├─ 試戴測試
   └─ 主導眼檢查（選配）

3. 處方確認（5 分鐘）
   ├─ 解釋度數意義
   ├─ 與舊眼鏡比較
   └─ 確認最終處方

4. 鏡框挑選（10-20 分鐘）
   ├─ 臉型分析
   ├─ 用途建議
   ├─ 試戴調整
   └─ 確認尺寸合適

5. 鏡片選擇（5-10 分鐘）
   ├─ 需求分析（抗藍光？變色？多焦？）
   ├─ 材質說明（樹脂、玻璃、PC）
   ├─ 鍍膜選項
   └─ 價格說明

6. 取片調整（取件時）
   ├─ 鏡框調整
   ├─ 瞳距確認
   ├─ 配戴教學
   └─ 適應期說明
```

## 三、你的差異化優勢

### 競品常見寫法（一般眼鏡行/連鎖店）

- 純流程條列，像說明書
- 強調設備多先進
- 沒有溫度，沒有故事
- 重點在「我們有什麼」

### 你可以不一樣

| 一般寫法 | 目鏡大叔的寫法 |
|----------|----------------|
| 「第一步：電腦驗光」 | 從一個緊張的客人走進門開始說起 |
| 「我們有最新儀器」 | 「儀器只是工具，重點是問對問題」 |
| 「驗光約 15 分鐘」 | 「有些人 15 分鐘，有些人我們會聊 1 小時」 |
| 「鏡框多種選擇」 | 「選鏡框像選衣服，不是越貴越好」 |

## 四、法規注意事項

### 必須提及的

1. **6 歲以下**：須由眼科醫師驗光或開立處方
2. **15 歲以下初次驗光**：須經眼科醫師確認非假性近視
3. **驗光人員法**：明確區分驗光師 vs 驗光生業務範圍

### 可以強調的專業點

- 驗光師 ≠ 店員（有國家證照）
- 驗光所 ≠ 眼鏡行（受衛生局管轄）
- 處方權的意義

### 建議角度

**角度 1：從「緊張的第一次」切入**
- 開場：一個緊張的媽媽帶孩子來配鏡的故事
- 帶出：其實你不用緊張，讓我告訴你會發生什麼事
- 一步步說明流程，穿插你會做的事（問診、解釋、不急著推銷）

**角度 2：從「為什麼我們比較慢」切入**
- 開場：有人問你為什麼驗光要這麼久
- 帶出：因為我在聽你的生活
- 解釋每個步驟背後的意義

**角度 3：從「常見誤解」切入**
- 開場：列出大家對配鏡的誤解
  - 「電腦驗光就是度數」
  - 「驗光師就是店員」
  - 「越貴的鏡片越好」
- 逐一破解，順便帶出正確流程

## 六、可引用的專業內容

### 驗光的專業環節

1. **電腦驗光 vs 綜合驗光**
   - 電腦驗光：粗估值，會受調節力影響
   - 綜合驗光：精確值，考慮雙眼協調

2. **為什麼要問生活習慣**
   - 辦公室 vs 戶外工作 → 影響鏡片選擇
   - 開車頻率 → 影響是否需要偏光/夜間鏡片
   - 3C 使用時間 → 影響是否需要抗疲勞設計

3. **適應期的科學解釋**
   - 新度數需要大腦適應
   - 通常 3-7 天
   - 什麼情況該回來調整

## 七、FAQ 建議

1. Q：配眼鏡要多久時間？
   A：驗光 + 選框約 30-60 分鐘，製作約 3-7 天（依鏡片種類）

2. Q：第一次配眼鏡要先去眼科嗎？
   A：成人可直接至驗光所；15 歲以下初次建議先眼科確認

3. Q：驗光會痛或不舒服嗎？
   A：完全不會，只是看視力表和回答問題

4. Q：可以只驗光不配鏡嗎？
   A：可以，我們提供獨立驗光服務

5. Q：為什麼不同店家驗出來度數不一樣？
   A：驗光方法、時間點、調節力控制都會影響
