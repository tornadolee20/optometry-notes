## 🎬 鏡頭配置

### Shot 1 (0-4秒) - 無法接受
**畫面**：
- 近景：男人早上起床，拿起手機
- 動作：手臂伸直，把手機拿遠
- 表情：皺眉，無奈

**指令**：
```
[SHOT 1: 室內/清晨]
[VISUAL: 近景男人側臉, 剛睡醒, 皺眉]
[ACTION: 伸手拿手機, 手臂慢慢伸直, 拿遠]
[EMOTION: 困惑, 不解]
[COLOR: 冷色調, 清晨微光]
[TRANSITION: 慢速推近]
```

**指令**：
```
[SHOT 2: 室內/驗光所]

[EMOTION: 震驚, 不願接受]

```

**指令**：
```
[SHOT 3: 室外/夜晚陽台]

[ACTION: 端茶杯, 輕輕嘆氣, 望向遠方]
[EMOTION: 釋然, 接受, 平静]
```
