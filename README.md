## 🛠️ 目前收錄工具

### [眼軸長度 (AL) 估算工具](./axial_length_calculator.html)
基於 2019-2024 年最新視光學文獻（Lingham, Morgan, Kim 等模型），透過等效球鏡度數 (SER) 與角膜曲率 (CR) 快速估算眼軸長度。
- **用途**：臨床風險分層、近視預後評估。
- **特性**：整合 AL/CR 比值警示，自動標註高風險對象。
