## 三、臨床評估標準

### 1. Morgan's Norms（摩根標準值）

| 測試項目 | 遠距離 | 近距離（40cm） |
|----------|--------|----------------|
| Phoria | 1 exo (±1) | 3 exo (±3) |
| BI break | 7 (±3) | 13 (±4) |
| BI recovery | 4 (±2) | 10 (±4) |
| BO break | 19 (±8) | 21 (±6) |
| BO recovery | 10 (±4) | 11 (±7) |
| NPC break | - | 5 cm (±2.5) |
| NPC recovery | - | 7 cm (±3) |

### 2. Sheard's Criterion（謝爾德準則）

> 補償性聚散力 ≥ 2 × Phoria 量

- 若不符合：可能出現視覺症狀
- 臨床應用：計算 BI/BO 儲備是否足夠

### 3. Percival's Criterion（珀西瓦爾準則）

> Phoria 應位於聚散範圍的中間 1/3

- 正向聚散:負向聚散比例評估
- 用於判斷舒適區

### 4. Vergence Facility（聚散靈活度）

| 條件 | 標準 |
|------|------|
| 測試用稜鏡 | 3Δ BI / 12Δ BO |
| 正常標準 | ≥15 cpm（近距離） |
| 失敗標準 | <15 cpm |
| 測試時間 | 1 分鐘 |

### 5. NPC（近點聚合）

| 狀態 | Break | Recovery |
|------|-------|----------|
| 正常 | ≤5-6 cm | ≤7-8 cm |
| CI 疑似 | >6 cm | >8 cm |
| CI 確定 | >10 cm | - |

## 四、常見雙眼視覺異常分類

### Non-Strabismic Binocular Vision Disorders (NSBVD)

| 類型 | 特徵 | 主要測試 |
|------|------|----------|
| **Convergence Insufficiency (CI)** | 近距離外斜位、NPC 後退、BO 不足 | NPC, Phoria, BO vergence |
| **Convergence Excess (CE)** | 近距離內斜位、AC/A 高 | Phoria, BI vergence, AC/A |
| **Divergence Insufficiency** | 遠距離內斜位 | Distance phoria, BI vergence |
| **Divergence Excess** | 遠距離外斜位 | Distance phoria, BO vergence |
| **Basic Exophoria** | 遠近皆外斜位 | All phorias |
| **Basic Esophoria** | 遠近皆內斜位 | All phorias |
| **Fusional Vergence Dysfunction** | 聚散範圍狹窄、斜位正常 | All vergences |

## 五、臨床檢查流程建議

### 標準雙眼視覺評估順序

```
1. 病史詢問
   ├─ 症狀問卷（CISS、COVD-QoL）
   ├─ 工作/用眼習慣
   └─ 過去配鏡/治療史

2. 基礎視覺功能
   ├─ 視力（遠/近）
   ├─ 屈光狀態
   └─ 眼位初步評估

3. 調節功能
   ├─ 調節幅度（Push-up / Minus lens）
   ├─ 調節靈活度（±2.00D flipper）
   ├─ 相對調節（NRA/PRA）
   └─ 調節反應（MEM/Nott）

4. 聚散功能
   ├─ Cover test（遠/近）
   ├─ Phoria 測量（Von Graefe / Maddox）
   ├─ NPC
   ├─ 聚散範圍（BI/BO, step/smooth）
   ├─ 聚散靈活度
   └─ AC/A 比值

5. 感覺融合
   ├─ Worth 4-dot
   ├─ Stereopsis（Titmus, Randot）
   └─ 抑制評估

6. 眼球運動
   ├─ 追視（Pursuits）
   └─ 跳視（Saccades）
```

## 六、症狀問卷工具

### 1. CISS（Convergence Insufficiency Symptom Survey）

- 15 題問卷
- 評分 0-4（從不 → 總是）
- 總分 ≥21：高度懷疑 CI
- 用途：篩檢 + 治療效果追蹤

### 2. COVD-QoL（College of Optometrists in Vision Development Quality of Life）

- 評估視覺相關生活品質
- 30 題完整版 / 19 題簡短版
