# 第一章：前言

---

## 1.1 台灣學童近視現況

台灣是全球近視盛行率最高的國家之一。根據教育部統計資料，我國學童的視力不良率隨年齡逐步攀升，至高中階段已超過八成。

### 各階段學生視力不良率（2024 年）

| 年級 | 視力不良率 |
|------|-----------|
| 國小一年級 | 25.9% |
| 國小六年級 | 61.5% |
| 國中三年級 | 76.0% |
| 高中三年級 | 83.7% |

### 近視率變化趨勢（2005-2024）

| 指標 | 2005 年 | 2011 年（高峰） | 2024 年 | 變化 |
|------|---------|----------------|---------|------|
| 小六視力不良率 | 54.8% | 65.0% | 61.5% | +6.7% |
| 國三視力不良率 | 71.6% | 77.7% | 76.0% | +4.4% |
| 高三視力不良率 | 86.0% | 87.7% | 83.7% | -2.3% |

**關鍵發現**：
- 2005 至 2011 年間，各階段近視率**持續惡化**
- 2011 年後才開始緩慢下降
- 至 2024 年，國小階段近視率仍高於 2005 年水準

### 高度近視盛行率

根據國民健康署 106 年調查（散瞳驗光）：

| 年級 | 高度近視率（≥500度） |
|------|---------------------|
| 國小六年級 | 10.3% |
| 國中三年級 | 28.0% |
| 高中三年級 | 35.7% |

**警訊**：超過三分之一的高中生已達高度近視，將面臨終生視力健康風險。

### 國際比較

| 國家/地區 | 青少年近視率 | 主要策略 |
|-----------|-------------|----------|
| 台灣 | ~85% | 散瞳劑為主 |
| 新加坡 | ~80% | 戶外活動 + 低濃度阿托品 |
| 中國（都市） | ~80% | 戶外活動 2 小時政策 |
| 澳洲 | ~20% | 自然戶外生活型態 |
| 美國 | ~40% | — |

## 1.2 現行政策檢討

### 1.2.1 散瞳劑政策回顧

民國 88 年（1999 年），在眼科醫學會主導下，台灣開始大規模推動以**散瞳劑（阿托品）** 作為學童近視控制的主要策略。

**政策內容**：
- 對近視學童點用 0.5% 或 1% 阿托品眼藥水
- 透過學校健康檢查發現異常後轉介眼科
- 作為國家級近視防治的核心措施

### 1.2.2 政策成效評估

**客觀數據分析**：

根據教育部 20 年的視力統計資料：

| 時期 | 小六視力不良率變化 | 評估 |
|------|-------------------|------|
| 1999-2005（政策推動前 6 年） | — | — |
| 2005-2011（政策推動中） | 54.8% → 65.0% | ❌ 惡化 10.2% |
| 2011-2024（政策調整後） | 65.0% → 61.5% | ⚠️ 略改善 3.5% |
| **整體 2005-2024** | **54.8% → 61.5%** | **❌ 仍惡化 6.7%** |

**結論**：政策推動超過 25 年，近視率**未見顯著下降**，反而在初期持續惡化。

### 1.2.3 問題分析

**為何高濃度散瞳劑政策成效不彰？**

| 問題 | 說明 |
|------|------|
| **副作用大** | 高濃度阿托品（0.5-1%）造成嚴重畏光、調節麻痺，影響日常學習 |
| **依從性差** | 副作用導致家長與學童不願持續使用 |
| **停藥反彈** | 停止用藥後近視可能加速進展 |
| **單一策略** | 未配合戶外活動等環境介入 |
| **環境惡化** | 3C 產品普及，近距離用眼時間大幅增加 |
| **缺乏追蹤** | 轉介後缺乏系統性追蹤機制 |
| **醫療量能不足** | 眼科診所塞車，檢查品質下降 |
| **專業人力錯置** | 驗光師資源未被有效運用 |

### 1.2.4 現行轉介制度的結構性問題

除了散瞳劑政策本身的問題外，現行學童視力保健體系還存在以下結構性問題：

#### 問題一：眼科塞車，醫療品質堪憂

每學期開學後，學校發出大量視力檢查回條，造成眼科診所大排長龍：
- 家長與學童動輒等待 **1-2 小時**
- 醫師被迫壓縮看診時間，甚至 **不到 5 分鐘**
- 無法提供完整的視機能評估與衛教
- 真正需要醫療的眼疾個案反而被排擠

#### 問題二：6 歲以下檢測的矛盾現象

《驗光人員法》規定驗光師不得對 6 歲以下兒童驗光，立法精神是保護幼童。然而：
- 眼科診所因人力不足，常由 **未具國考證照的人員** 操作驗光儀器
- 形成「**法規最嚴、現場最鬆**」的矛盾
- 最需要專業評估的族群，反而得到最不專業的服務

> **有法規沒人力，有名目沒實質**——這是台灣學童視力保健體系的核心困境。

### 1.2.5 國際實證的啟示

根據亞洲大型臨床研究（如 LAMP Study）及多項統合分析：

1. **0.05% 阿托品是目前的黃金標準**——效果接近高濃度，副作用大幅減少
2. **戶外活動可減少 46% 的新發近視**
3. **多種介入措施（光學、藥物、環境）應綜合運用**

**新加坡經驗**：
- 推動「戶外活動 + 低濃度阿托品」雙軌策略
- 近視進展速度明顯減緩

## 1.3 本手冊目的與定位

### 1.3.1 目的

本手冊由**臺灣視光視力保健學會**編撰，旨在：

1. **提供實證為基礎的建議**：所有建議均引用高品質研究證據
2. **客觀評估現行政策**：以數據檢視政策成效
3. **提出更有效的策略**：參考國際經驗與最新研究
4. **建立可執行的方案**：在現行體制內可落地執行

### 1.3.2 定位

| 項目 | 說明 |
|------|------|
| **發起單位** | 臺灣視光視力保健學會 |
| **適用對象** | 學校、衛生單位、教育主管機關、家長 |
| **適用範圍** | 高級中等以下學校（幼兒園至高中職） |
| **專業角度** | 驗光師專業，配合眼科醫療體系 |

### 1.3.3 本手冊的核心主張

1. ✅ **預防重於治療**：強調戶外活動等環境介入
2. ✅ **實證為本**：所有建議皆有研究支持
3. ✅ **綜合策略**：不依賴單一方法
4. ✅ **跨專業合作**：驗光師、眼科、學校共同照護
5. ✅ **可執行性**：在現行法規與體制內可落實
6. ✅ **全齡視野**：從學童近視延伸至全民視力健康
7. ✅ **城鄉均衡**：關注偏鄉醫療資源不足的問題
3. ✅ **綜合策略**：不依賴單一方法
