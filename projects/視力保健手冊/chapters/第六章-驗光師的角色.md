# 第六章：驗光師的角色

---

## 本章重點

本章說明驗光師在學童視力保健中的專業角色、法定業務範圍，以及如何與眼科醫療體系合作。

---

## 6.1 驗光人員法規定

### 6.1.1 驗光師業務範圍

根據**驗光人員法第 12 條**，驗光師的業務範圍包括：

1. **非侵入性之眼球屈光狀態測量及相關驗光**
   - 包含為一般隱形眼鏡配鏡所為之驗光
   - **15 歲以下者應於眼科醫師指導下為之**
   - **未滿 6 歲兒童之驗光，不得為之**

2. **一般隱形眼鏡之配鏡**

3. **低視力者輔助器具之教導使用**

4. **其他依醫師開具之照會單或醫囑單所為之驗光**

### 6.1.2 驗光生業務範圍

根據**驗光人員法第 13 條**，驗光生的業務範圍包括：

1. **一般性近視、遠視、散光及老花之驗光**
   - **15 歲以下者應於眼科醫師指導下為之**
   - **未滿 6 歲兒童之驗光，不得為之**

2. **一般隱形眼鏡之配鏡**

3. **其他依醫師開具之照會單或醫囑單所為之驗光**

### 6.1.3 年齡限制摘要

| 年齡 | 驗光師 | 驗光生 |
|------|--------|--------|
| **< 6 歲** | ❌ 不可執行 | ❌ 不可執行 |
| **6-15 歲** | ⚠️ 需眼科醫師指導 | ⚠️ 需眼科醫師指導 |
| **≥ 15 歲** | ✅ 可獨立執行 | ✅ 可獨立執行 |

## 6.2 驗光師可以做什麼

### 6.2.1 屈光檢查

| 項目 | 說明 |
|------|------|
| 電腦驗光 | 初步客觀屈光測量 |
| 檢影驗光 | 精確客觀屈光測量 |
| 主觀驗光 | 確認最佳矯正度數 |
| 調節測量 | 評估調節幅度與靈活度 |
| 聚散測量 | 評估雙眼協調功能 |

### 6.2.2 雙眼視覺評估

| 項目 | 說明 |
|------|------|
| 遮蓋測試 | 評估斜視與隱斜視 |
| 融像能力測量 | 評估雙眼融像功能 |
| 立體視測試 | 評估深度知覺 |
| 調節與聚散關係 | 評估雙眼協調 |

### 6.2.3 配鏡服務

| 項目 | 說明 |
|------|------|
| 一般眼鏡配戴 | 近視、遠視、散光、老花矯正 |
| 多焦眼鏡 | 漸進、雙光鏡片驗配 |
| **近視控制鏡片** | 周邊離焦鏡片驗配 |
| 隱形眼鏡 | 一般軟性、硬性隱形眼鏡驗配 |
| **角膜塑型** | 需專業訓練與認證 |

### 6.2.4 近視管理服務

| 項目 | 說明 |
|------|------|
| 近視風險評估 | 遠視儲備、眼軸測量、家族史評估 |
| 控制方案諮詢 | 說明各種控制選項 |
| 光學介入執行 | 角膜塑型、近視控制鏡片驗配 |
| 定期追蹤 | 監測度數與眼軸變化 |
| 轉介建議 | 需藥物治療時轉介眼科 |

### 6.2.5 低視力服務

| 項目 | 說明 |
|------|------|
| 評估 | 評估低視力者的視覺功能 |
| 輔具教導 | 教導使用放大鏡、望遠鏡等輔具 |
| 環境建議 | 提供照明與環境調整建議 |

---

## 6.3 驗光師不能做什麼

| 項目 | 說明 | 應轉介 |
|------|------|--------|
| 眼疾診斷 | 如白內障、青光眼、視網膜病變等 | 眼科 |
| 眼藥水處方 | 包含阿托品等散瞳劑 | 眼科 |
| 6 歲以下驗光 | 法律明文禁止 | 眼科 |
| 侵入性檢查 | 如眼壓測量（部分）、眼底攝影等 | 眼科 |

## 6.4 與眼科的分工合作

### 6.4.1 合作模式

```
學童視力保健
├── 預防階段
│   ├── 驗光師：屈光篩檢、風險評估、衛教
│   └── 眼科：6 歲以下檢查、疾病排除
│
├── 發現異常
│   ├── 驗光師：精密驗光、雙眼視覺評估
│   └── 眼科：散瞳驗光、疾病診斷
│
├── 近視控制
│   ├── 驗光師：光學介入（角膜塑型、近視控制鏡片）
│   └── 眼科：藥物治療（阿托品處方）
│
└── 追蹤管理
    ├── 驗光師：定期屈光追蹤、眼軸監測
    └── 眼科：併發症篩檢、藥物調整
```

### 6.4.2 轉介時機

#### 驗光師應轉介眼科的情況

| 情況 | 說明 |
|------|------|
| 6 歲以下兒童 | 法律規定 |
| 疑似眼疾 | 如紅眼、疼痛、飛蚊症、閃光等 |
| 需要散瞳驗光確認 | 假性近視排除 |
| 需要阿托品處方 | 藥物治療 |
| 高度近視者 | 需定期眼底檢查 |
| 視力異常無法解釋 | 可能有器質性病變 |

#### 眼科可轉介驗光師的情況

| 情況 | 說明 |
|------|------|
| 配鏡需求 | 詳細驗光與配鏡 |
| 近視控制 | 角膜塑型、近視控制鏡片驗配 |
| 雙眼視覺異常 | 詳細雙眼視覺評估 |
| 低視力服務 | 輔具評估與教導 |

### 6.4.3 共同照護的優勢

| 優勢 | 說明 |
|------|------|
| 專業分工 | 各司其職，發揮專長 |
| 服務可近性 | 驗光所分布廣泛，增加服務據點 |
| 減少眼科負擔 | 讓眼科專注於疾病診療 |
| 持續追蹤 | 驗光師可提供更頻繁的追蹤服務 |
| 完整照護 | 預防、控制、追蹤一條龍 |

| 回報機制 | 檢查結果回報學校 |
| 追蹤協助 | 提供定期追蹤服務 |

### 6.5.3 衛教支援服務

| 項目 | 說明 |
|------|------|
| 學生衛教 | 到班級進行視力保健講座 |
| 家長衛教 | 配合親師座談進行衛教 |
| 教師研習 | 提供教師相關知能培訓 |

### 6.5.4 建議的法規調適

| 現行限制 | 建議方向 |
|---------|---------|

| 偏鄉眼科資源不足 | 建立遠距醫療指導機制 |

## 6.6 驗光師的社會責任與公衛功能

驗光師的角色不應僅限於「配眼鏡」，而應被定位為**社區視力健康的第一線守門員**。這個定位能幫助政府解決兩大難題：高齡化社會的照護需求，以及城鄉醫療資源不均。

### 6.6.1 填補眼科「衛教真空帶」：全齡視力諮詢

#### 現況分析

眼科醫療高度專業化且業務繁忙，醫師的主力在於**「診斷與治療疾病」**：
- 白內障手術
- 青光眼治療
- 視網膜病變處置
- 各類眼疾診斷

然而，許多民眾需要的是**「生活化的視力諮詢與衛教」**，這些服務眼科醫師難以撥出大量時間提供：

| 族群 | 需要的服務 | 眼科現況 |
|------|-----------|---------|
| 學童 | 正確用眼姿勢、閱讀距離、光線調整 | 看診時間短，難以細緻衛教 |
| 上班族 | 電腦視覺症候群、乾眼症預防 | 輕症常被忽略 |
| 中高齡 | 老花眼適應、多焦鏡片使用教導 | 無暇詳細說明 |
| 低視力者 | 輔具使用、環境調整、生活適應 | 需專門時間，眼科難以兼顧 |
| 高齡長者 | 視覺引發的跌倒預防 | 幾乎無人提供此服務 |

#### 驗光師的角色

**驗光師應被定位為「視力衛教的主要執行者」**：

| 服務對象 | 驗光師可提供的服務 |
|---------|------------------|
| **學童** | 用眼習慣指導、閱讀環境評估、近視控制諮詢 |
| **青少年** | 隱形眼鏡衛教、運動視覺需求評估 |
| **成年人** | 職業視覺需求評估、電腦視覺症候群預防 |
| **中高齡** | 老花眼解說、漸進鏡片適應指導、多焦鏡片配戴訓練 |
| **低視力者** | 輔具評估與使用教導（法定業務）、生活環境調整建議 |
| **高齡長者** | 視覺相關跌倒預防、照明環境建議、輔具諮詢 |

#### 高齡視覺與跌倒預防

這是目前醫療體系中**幾乎無人關注的領域**，卻與高齡者的生命安全密切相關：

| 視覺問題 | 跌倒風險 |
|---------|---------|
| 老花未矯正 | 看不清階梯邊緣 |
| 多焦鏡片不適應 | 下樓梯時視覺混亂 |
| 對比敏感度下降 | 難以辨識地面障礙物 |
| 周邊視野缺損 | 未察覺側方來物 |
| 暗適應變慢 | 從亮處進入暗處時失足 |

**驗光師可以做的**：
- 評估視覺功能與跌倒風險的關聯
- 教導多焦鏡片的正確使用（尤其上下樓梯）
- 建議居家照明改善
- 必要時轉介眼科或其他專科

> **眼科醫師很忙，沒時間教阿公阿嬤「樓梯要怎麼看才不會摔倒」——這正是驗光師的專業！**

---

### 6.6.2 偏鄉視力守護：驗光所作為「社區前哨站」

#### 現況分析

台灣醫療資源城鄉差距嚴重，眼科尤其明顯：

| 地區 | 眼科醫師分布 | 民眾就醫情況 |
|------|-------------|-------------|
| 都會區 | 密集 | 就近可得 |
| 一般鄉鎮 | 少量 | 需跨區就醫 |
| **偏遠鄉鎮** | **極少或無** | **就醫成本極高** |

**然而**：

| 醫療資源 | 分布情況 |
|---------|---------|
| 眼科診所 | 集中於都會區 |
| **驗光所/眼鏡行** | **深入社區與鄰里，偏鄉亦有分布** |

驗光所的分布密度遠高於眼科診所，且營業時間更彈性、更貼近民眾生活。

#### 建議：驗光所作為「初級視力篩檢站」

```
偏鄉民眾視力問題
    ↓
在地驗光所（社區前哨站）
    ├─→ 單純屈光問題 → 在地解決（驗光配鏡）
    │
    └─→ 疑似眼疾 → 轉介至區域醫院眼科
                    ↓
              確診後回到社區追蹤
```

**具體建議**：

| 建議事項 | 說明 |
|---------|------|
| **賦予篩檢功能** | 政府應正式認可驗光所作為「初級視力篩檢站」 |
| **建立轉介機制** | 驗光所與區域醫院眼科建立轉介合作 |
| **遠距指導機制** | 透過遠距醫療，讓偏鄉驗光師可獲得眼科醫師指導 |
| **納入長照體系** | 驗光師可參與社區長照的視力評估服務 |
| **巡迴服務補助** | 政府補助驗光師至偏鄉進行視力巡迴服務 |

#### 預期效益

| 效益 | 說明 |
|------|------|
| **減少偏鄉就醫障礙** | 一般屈光問題在地解決 |
| **早期發現眼疾** | 驗光師可篩出疑似個案並轉介 |
| **落實在地老化** | 長者不必遠赴都會區就醫 |
| **活化在地專業人力** | 偏鄉驗光師有更多發揮空間 |
| **節省醫療資源** | 減少不必要的跨區就醫 |

### 6.6.3 驗光師的公衛定位：不只是配眼鏡

**傳統認知**：驗光師 = 眼鏡店店員

**專業定位**：驗光師 = **視力健康的第一線守門員** + **眼科體系的深度補位者**

| 角色 | 功能 |
|------|------|
| **篩檢者** | 初步發現視力異常，轉介就醫 |
| **矯正者** | 精準驗光，光學矯正屈光不正 |
| **衛教者** | 提供用眼習慣、視覺環境的專業指導 |
| **追蹤者** | 長期監測屈光變化，評估控制成效 |
| **諮詢者** | 回答民眾視力相關問題，提供第二意見 |
| **連結者** | 銜接眼科醫療與社區需求 |

### 我們的格調

> **我們不搶那張紙（回條），我們搶的是『家長的信任』和『孩子的視力健康』。**

> **驗光師不是只會賣眼鏡，我們是填補醫療空窗期的水泥。**

> **有我們在，大醫院不會塞爆，偏鄉的阿公阿嬤也不會求助無門。**

> **眼科做不到的精緻服務，就是我們的戰場。這才是驗光師的格調。**
