# 第四章：近視控制方法

---

## 本章重點

當近視已經發生，目標轉為**控制進展**，延緩度數增加與眼軸增長，降低未來高度近視的風險。

本章介紹目前有實證支持的近視控制方法，並提供客觀的比較。

---

## 4.1 藥物介入

### 4.1.1 阿托品眼藥水（Atropine）

#### 作用機制

阿托品是一種**非選擇性蕈毒鹼受體拮抗劑**，可透過以下機制減緩近視進展：
- 作用於視網膜或鞏膜的蕈毒鹼受體
- 可能影響眼軸增長的訊號傳遞
- 確切機制仍在研究中

#### 不同濃度比較

根據亞洲地區大型臨床研究（LAMP Study, 香港中文大學）及多項網絡統合分析（Ha et al. 2022）：

| 濃度 | 1 年屈光控制 (D) | 1 年眼軸控制 (mm) | 副作用 | 備註 |
|------|-----------------|------------------|--------|------|
| **1%** | 0.81 | -0.35 | 嚴重 | 效果最強，但副作用大 |
| **0.5%** | 0.70 | -0.23 | 明顯 | 台灣過去主流 |
| **0.05%** | 0.62 | -0.25 | 輕微 | ⭐ **目前黃金標準** |
| **0.01%** | 0.38 | -0.13 | 極輕 | 效果較弱，歐美研究效果不一致 |

**重要發現**：

根據 LAMP Study（2019-2022）的研究結果：
- **0.05%** 在控制眼軸增長上效果顯著，且副作用極低
- **0.01%** 的效果在不同研究中結果不一致（亞洲研究效果較佳，歐美研究效果較差）
- 目前國際趨勢認為 **0.05% 是最佳平衡點（Sweet Spot）**

#### 副作用

| 濃度 | 畏光 | 調節麻痺 | 瞳孔放大 | 近距離視力影響 |
|------|------|---------|---------|---------------|
| 高濃度（0.5-1%） | 嚴重 | 嚴重 | 明顯 | 需配戴近用眼鏡 |
| 中濃度（0.1-0.25%） | 中等 | 中等 | 中等 | 可能需要 |
| **低濃度（0.01-0.05%）** | 輕微 | 極輕 | 輕微 | 通常不需要 |

#### 停藥反彈效應

研究顯示，停用阿托品後可能出現近視進展加速的「反彈效應」：
- 高濃度反彈較明顯
- 低濃度反彈較輕
- 建議逐步減量停藥，而非突然停止

#### ⚠️ 雙眼視機能警語

**重要提醒**：長期使用散瞳劑（尤其是較高濃度）可能影響**近見三連動（Near Triad）**機制，包括：
- **調節（Accommodation）**：看近時水晶體變厚的能力
- **輻輳（Convergence）**：雙眼向內轉動對準近物的能力
- **瞳孔收縮（Miosis）**：看近時瞳孔縮小的反應

這三者在正常視覺中協同運作。散瞳劑會抑制調節與瞳孔收縮，進而可能影響 **AC/A ratio（調節性輻輳與調節比值）**，導致雙眼視覺協調問題。

**臨床建議**：
- 使用散瞳劑的學童應**定期監測雙眼視功能**
- 評估項目包括：調節幅度、調節靈活度、輻輳近點（NPC）、融像範圍
- 如發現雙眼視覺異常，應轉介驗光師進行完整評估

#### 臨床建議

| 項目 | 建議 |
|------|------|
| 建議濃度 | **0.05%**（目前國際共識的最佳濃度） |
| 次選濃度 | 0.01%（副作用更低，但效果較不穩定） |
| 使用頻率 | 每晚一次 |
| 使用年齡 | 6 歲以上 |
| 使用期間 | 至少 2 年，視情況調整 |
| 停藥方式 | 逐步減量 |
| **處方權** | **需由眼科醫師處方** |

#### 台灣現況問題

| 問題 | 說明 |
|------|------|
| 過去使用高濃度為主 | 副作用大，依從性差 |
| 低濃度製劑取得 | 需自費、藥局調劑 |
| 缺乏長期追蹤 | 停藥後缺乏系統性監測 |

## 4.2 光學介入

### 4.2.0 基礎矯正優先原則：回歸單焦點鏡片

在談論各種近視控制鏡片之前，我們必須先強調一個經常被忽視的重要原則：

> **不應跳過「基礎矯正」這一階。**

#### 單焦點鏡片（Single Vision Lenses, SVL）的角色

對於**初次發現近視**或**度數穩定**的學童，**足度矯正的單焦點鏡片**仍是第一線的標準配備。

| 情境 | 建議 |
|------|------|
| 初次發現近視（< 100 度） | 先配單焦點鏡片觀察 |
| 度數穩定（< 50 度/年） | 單焦點鏡片可能已足夠 |
| 假性近視或調節緊張 | 配正確度數後可能穩定 |
| 快速進展（> 75-100 度/年） | 考慮升級控制方案 |

**為什麼要先從單焦點開始？**

1. **消除視網膜模糊**：讓眼球在清晰成像的環境下發育
2. **觀察基線反應**：了解孩子在正常矯正下的進展速度
3. **避免過度醫療**：很多孩子其實不需要昂貴的控制鏡片
4. **節省家長荷包**：不必一開始就花兩三萬

#### ⏰ 黃金法則：3 個月密集追蹤期

配戴單焦點鏡片後，**絕非「半年後見」**。

**必須設定 3 個月為黃金追蹤期**：

```
配戴單焦點鏡片 + 行為改變（多戶外、少近業）
    ↓
【3 個月後追蹤】
    ├─→ 度數/眼軸無明顯變化
    │     ✅ 單焦點鏡片有效，繼續維持
    │     ✅ 家長省荷包，孩子也輕鬆
    │
    └─→ 度數/眼軸快速增加
          ⚠️ 單焦點「控制力不足」
          ➡️ 此時才有正當理由升級處置
```

#### 追蹤標準

| 3 個月追蹤結果 | 判斷 | 建議處置 |
|--------------|------|---------|
| 度數增加 < 25 度 | 穩定 | 繼續單焦點，6 個月後追蹤 |
| 度數增加 25-50 度 | 邊緣 | 加強行為控制，3 個月再追蹤 |
| **度數增加 > 50 度** | **快速進展** | **升級為近視控制方案** |
| 眼軸增長 > 0.1 mm | 警戒 | 評估升級處置 |

#### 這才是有良心的做法

> **不要被廣告嚇到了。**
> 
> 孩子剛近視，是不是一定要馬上配那種一副兩三萬的控制鏡片？
> 
> **未必！**
> 
> 很多孩子，其實只需要一副配得準、戴得舒服的普通眼鏡，加上改掉趴著寫字的壞習慣，度數就穩住了。

**我們的做法**：
1. 先配單焦點鏡片（精準驗光、足度矯正）
2. 約法三章：**3 個月後一定要回來追蹤**
3. 如果 3 個月後度數沒變 → 繼續戴普通的，省錢又有效
4. 如果 3 個月後擋不住 → 再談升級裝備，錢花在刀口上

> **這才是有良心的驗光師該做的事，不是一見面就把家長當肥羊宰。**

### 4.2.1 角膜塑型（Orthokeratology, Ortho-K）

#### 原理

- 夜間配戴特殊設計的硬式隱形眼鏡
- 暫時改變角膜弧度
- 白天不需戴眼鏡即可看清
- 同時產生周邊離焦效果，抑制眼軸增長

#### 效果

| 指標 | 效果 |
|------|------|
| 眼軸控制（1 年） | 減少約 **0.19 mm** |
| 眼軸控制（2 年） | 減少約 **0.28 mm** |
| 近視控制效果 | 約 **40-60%** |

#### 優缺點

| 優點 | 缺點 |
|------|------|
| 白天不需戴眼鏡 | 成本較高 |
| 兼具矯正與控制效果 | 需每晚配戴 |
| 可逆（停戴後恢復） | 有感染風險（需嚴格清潔） |
| 適合運動兒童 | 需定期回診追蹤 |

#### 適用對象

- 年齡：**8 歲以上**
- 近視度數：通常 **< 500 度**
- 散光：通常 **< 150 度**
- 能配合夜間配戴與清潔保養
- 無角膜疾病

#### 風險與注意事項

| 風險 | 預防措施 |
|------|---------|
| 角膜感染 | 嚴格清潔、定期更換鏡片 |
| 角膜缺氧 | 選擇高透氧材質 |
| 鏡片移位 | 定期追蹤調整 |

**重要**：需由**合格驗光師或眼科醫師**驗配，並定期追蹤。

---

### 4.2.2 周邊離焦眼鏡（Defocus Incorporated Multiple Segments, DIMS / HAL）

#### 原理

- 鏡片設計在周邊產生近視性離焦
- 減緩眼軸增長的訊號
- 外觀與普通眼鏡相似

#### 效果

| 產品類型 | 近視控制效果 |
|---------|-------------|
| DIMS（如 MiYOSMART） | 約 **50-60%** |
| HAL（如 Stellest） | 約 **50-60%** |

#### 優缺點

| 優點 | 缺點 |
|------|------|
| 外觀如普通眼鏡 | 價格較高 |
| 無侵入性 | 需全天配戴才有效 |
| 適合各年齡層 | 部分兒童可能不適應 |
| 無感染風險 | 效果可能因人而異 |

#### ⚠️ 重要提醒：仿單限制問題

**目前部分周邊離焦鏡片廠商在產品仿單（使用說明書）中特意載明「應先散瞳驗光」**。

這造成以下問題：

| 問題 | 說明 |
|------|------|
| **驗光師無法獨立驗配** | 散瞳為醫療行為，需眼科醫師執行 |
| **形成人為障礙** | 實質上將驗光師排除在這類鏡片的驗配流程之外 |
| **增加家長負擔** | 需先至眼科散瞳，再至驗光所配鏡，流程繁瑣 |

**學理討論**：

| 觀點 | 說明 |
|------|------|
| 支持散瞳 | 排除假性近視、確認真實度數 |
| 質疑必要性 | 周邊離焦鏡片並非藥物或侵入性治療，是否需要強制散瞳？ |
| 現實考量 | 一般單焦點近視眼鏡不需強制散瞳，為何控制鏡片需要？ |

**驗光師的立場**：

1. 我們尊重法規與仿單的規範
2. 但對於「是否所有近視控制鏡片都必須強制散瞳」這一問題，學理上仍有討論空間
3. 建議廠商與專業團體重新檢視仿單內容的合理性
4. **在現行規範下，驗光師應依仿單指示，建議家長先至眼科散瞳驗光**

> **實務建議**：配戴周邊離焦鏡片前，請確認產品仿單的要求。若仿單載明需散瞳，應建議家長先至眼科完成散瞳驗光，再進行後續配鏡。

#### 適用對象

- 年齡：**6 歲以上**
- 各度數皆可
- 不適合或不願配戴隱形眼鏡者

---

### 4.2.3 多焦軟性隱形眼鏡

#### 原理

- 特殊設計的軟性隱形眼鏡
- 中央矯正近視，周邊產生離焦
- 日戴型

#### 效果

| 指標 | 效果 |
|------|------|
| 屈光控制（1 年） | 約 **0.26 D** |
| 眼軸控制（1 年） | 減少約 **0.11 mm** |
| 近視控制效果 | 約 **30-50%** |

#### 適用對象

- 年齡：**8 歲以上**
- 能配合日間配戴與清潔
- 無嚴重乾眼或過敏

---

### 4.2.4 多焦眼鏡（漸進多焦或雙焦）

#### 原理

- 減少看近時的調節負擔
- 理論上可減緩近視進展

#### 效果

| 指標 | 效果 |
|------|------|
| 屈光控制（1 年） | 約 **0.14 D** |
| 眼軸控制（1 年） | 減少約 **0.06 mm** |
| 近視控制效果 | 約 **10-20%**（較弱） |

#### 適用對象

- 合併**調節功能異常**者效果較佳
- 高調節滯後（high lag of accommodation）
- 內斜位傾向

**注意**：對一般近視兒童效果有限，不作為首選。

### 4.3.3 選擇建議

| 情境 | 建議方法 |

| 預算有限 | 低濃度阿托品 + 普通眼鏡 |

| 合併調節問題 | 多焦眼鏡或視覺訓練 |

## 4.4 合併治療

近年研究顯示，**合併使用**不同機制的控制方法可能有加成效果：

| 組合 | 說明 |
|------|------|
| 阿托品 + 角膜塑型 | 初步研究顯示可能有加成效果 |
| 阿托品 + 周邊離焦眼鏡 | 理論上可行，研究進行中 |

**注意**：合併治療需在專業人員指導下進行。

---

## 4.5 不建議的做法

| 做法 | 原因 |
|------|------|
| **欠矯（配不足的度數）** | 研究證實可能**加速**近視進展 |
| 完全不配眼鏡 | 無法減緩進展，且影響學習 |
| 僅靠戶外活動控制 | 對已近視者效果有限 |
| 未經驗證的療法 | 浪費時間與金錢，延誤治療 |

## 本章重點摘要

| 項目 | 要點 |
|------|------|
| 藥物首選 | **0.05% 阿托品** — 目前國際共識的最佳濃度，效果佳、副作用低 |
| 光學首選 | **角膜塑型**或**周邊離焦眼鏡** — 依個人需求選擇 |
| 效果有限 | 多焦眼鏡（除非有調節問題）、戶外活動（控制效果有限） |
| 不建議 | 欠矯、不配眼鏡、未經驗證療法 |
| 快速進展者 | 考慮合併治療 |
| 重要提醒 | 阿托品需眼科處方；角膜塑型需專業驗配與追蹤 |
