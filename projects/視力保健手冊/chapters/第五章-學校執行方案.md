# 第五章：學校視力保健執行方案

---

## 本章重點

本章提供學校可實際執行的視力保健方案，包括篩檢流程、轉介機制、健康教育與環境改善。

---

## 5.1 視力篩檢流程

### 5.1.1 篩檢目的

1. **早期發現**視力異常學生
2. **及時轉介**就醫處理
3. **追蹤管理**視力變化
4. **健康教育**的切入點

### 5.1.2 篩檢時機

| 時機 | 對象 | 目的 |
|------|------|------|
| **入學篩檢** | 新生 | 建立基線資料 |
| **每學期定期篩檢** | 全校學生 | 監測視力變化 |
| **學期中追蹤** | 視力異常學生 | 確認就醫與改善情形 |

### 5.1.3 篩檢方法

#### 標準視力表檢查

| 項目 | 標準 |
|------|------|
| 檢查距離 | **6 公尺**（或依視力表規格） |
| 照明 | 視力表照度 **≥ 500 lux** |
| 遮眼方式 | 使用遮眼板，避免壓迫眼球 |
| 檢查順序 | 先右眼，後左眼 |
| 記錄方式 | 十進位法（如 0.8、1.0） |

#### 視力檢查 SOP

1. 學生站立於指定距離
2. 以遮眼板遮住一眼（不可按壓）
3. 請學生辨認視力表上的符號或字母
4. 由大至小，記錄最小可辨識行
5. 換眼重複步驟
6. 記錄雙眼視力

### 5.1.4 異常標準

| 視力值 | 判定 | 處置 |
|--------|------|------|
| ≥ 1.0 | 正常 | 維持追蹤 |
| 0.9 | 邊緣 | 加強衛教，下次追蹤 |
| **≤ 0.8** | **異常** | **發通知單，建議就醫** |
| 雙眼差異 ≥ 2 行 | 異常 | 建議就醫 |

### 5.1.5 篩檢人員

| 現行做法 | 建議改進 |
|---------|---------|
| 護理師執行 | 可維持 |
| 導師協助 | 可維持 |
| - | **建議納入驗光師到校服務** |

**驗光師到校服務的優勢**：
- 更專業的屈光篩檢
- 可提供初步屈光狀態評估
- 更準確的轉介建議

## 5.2 轉介機制

### 5.2.1 轉介標準

| 情況 | 處置 |
|------|------|
| 裸視 ≤ 0.8 | 發通知單，建議就醫檢查 |
| 視力突然下降 | 儘速就醫 |
| 近視快速增加（> 50 度/學期） | 建議眼科評估控制方案 |
| 其他視覺異常（斜視、複視等） | 儘速轉介眼科 |

### 5.2.2 轉介流程

```
視力篩檢異常
    ↓
發送「視力檢查通知單」給家長
    ↓
家長帶學生至眼科或驗光所檢查
    ↓
將「回條」繳回學校
    ↓
護理師登錄追蹤
    ↓
下次篩檢時確認改善情形
```

### 5.2.3 就醫管道

| 管道 | 適用情況 |
|------|---------|
| **眼科診所/醫院** | 需要散瞳驗光、藥物治療、疾病診斷 |
| **驗光所** | 屈光檢查、配鏡、隱形眼鏡驗配 |

**注意**：
- 6 歲以上兒童可至驗光所檢查
- 15 歲以下驗光需有眼科醫師指導
- 6 歲以下需至眼科檢查

### 5.2.4 追蹤機制

| 項目 | 做法 |
|------|------|
| 回條回收 | 確認學生已就醫 |
| 未就醫追蹤 | 電話聯繫家長，了解原因 |
| 定期複檢 | 視力異常學生每學期追蹤 |
| 資料記錄 | 建立個人視力健康檔案 |

## 5.2.5 現行轉介機制的困境與改革建議

### 困境一：眼科塞車，品質堪憂

#### 現況描述

每學期開學後，學校發出大量「視力檢查回條」，要求家長帶學童就醫複查。這造成以下問題：

| 問題 | 實際情況 |
|------|---------|
| **眼科診所大排長龍** | 開學第一週，單一診所一天湧入數十名學童 |
| **等待時間過長** | 家長與學童動輒等待 1-2 小時 |
| **看診時間壓縮** | 醫師被迫縮短單次看診時間，甚至不到 5 分鐘 |
| **檢查品質下降** | 無法進行完整的視機能評估與衛教 |
| **家長困惑離開** | 拿著回條蓋了章，卻不清楚孩子眼睛到底怎麼了 |

**結果**：需要眼科醫師專業處理的真正眼疾（如青光眼、視網膜病變、斜弱視）反而被排擠。

#### 驗光師的立場：不搶回條，專注精緻服務

**我們的態度非常明確**：

> **學校的視力回條，我們驗光師一點興趣都沒有。**

為什麼？

| 原因 | 說明 |
|------|------|
| **違反專業良知** | 「排隊 1 小時、看診 3 分鐘、蓋個章」的模式，無法提供真正的照護 |
| **不願犧牲品質** | 我們不想為了掛號費，犧牲幫孩子仔細檢查的時間 |
| **定位不同** | 我們是「精品店」，不是「流水線」 |

那種「蓋章比賽」的工作，就留在現行的醫療體系裡。

**我們不背書那種 3 分鐘快速檢查的品質。**

├── 「度數增加那麼快，日常生活要怎麼改善？」
└── 「角膜塑型、近視控制鏡片，到底該怎麼選？」
    ↓

| **光學矯正執行** | 角膜塑型驗配、近視控制鏡片配戴 | 依方案 |

**這些都是眼科醫師「沒時間做」也「不想做」的高端、耗時服務。**

#### 差異化定位

|--------------------|------------------|

### 呼籲：眼科體系應正視品質問題

我們無意介入回條的行政流程，但必須指出：

**現行「回條—眼科—蓋章」的模式，存在嚴重的品質隱憂。**

| 問題 | 說明 |
|------|------|
| 檢查流於形式 | 許多家長反映「看診不到 5 分鐘」 |
| 衛教嚴重不足 | 家長離開時仍然一頭霧水 |
| 追蹤機制薄弱 | 蓋完章就結束，缺乏後續管理 |

**我們的建議**：

1. 眼科體系應自我檢視回條檢查的品質
2. 若人力不足，應誠實評估服務量能
3. 對於需要深度諮詢的個案，可主動告知家長尋求驗光師協助

**我們不是來搶生意的，我們是來補位的。**

### 困境二：6 歲以下檢測，「法規最嚴、現場最鬆」的矛盾

#### 現況描述

《驗光人員法》規定：

> 「**未滿 6 歲兒童之驗光，不得為之。**」

這條法規的立法精神是**保護幼童**，因為 6 歲以下兒童的視覺系統尚在發育，需要更專業、更謹慎的評估。

**然而現實卻是**：

| 法規精神 | 實際現況 |
|---------|---------| 
| 6 歲以下應由眼科醫師親自檢查 | 眼科醫師人力不足，難以親力親為 |
| 應由最專業的人員執行 | 常由未具國考證照的護理人員或助理操作驗光儀器 |
| 保護幼童視力健康 | 反而是專業度最低的環境 |

---

### ⚠️ 核心主張：6 歲以下驗光，眼科醫師必須「親力親為」

#### 法理邏輯的矛盾

這裡存在一個**無法迴避的法理矛盾**：

《驗光人員法》禁止驗光師對 6 歲以下兒童驗光，理由是：
- 6 歲以下眼球發育尚未成熟
- 檢查具有極高的醫療專業性與風險
- 必須由醫師介入

**如果這個邏輯成立，那麼**：

| 前提 | 推論 |
|------|------|
| 6 歲以下驗光具有「極高專業性與風險」 | **必須由最專業的人員執行** |
| 連通過國家高考的驗光師都被「排除在外」 | **代表這項工作的專業門檻比驗光師更高** |
| 唯一高於驗光師的專業 | **只有眼科醫師本人** |

**因此，依照法規的邏輯**：

> **6 歲以下兒童的驗光，只能由眼科醫師「親自」執行。**
> 
> **不是「在旁指導」，不是「最後判讀」，而是「親手操作」。**

#### 現況的荒謬

但實際上，診所內的情況往往是：

| 角色 | 實際工作 |
|------|---------|
| 眼科醫師 | 坐在診間，看報表、下結論 |
| 診所助理（無驗光證照） | 操作電腦驗光機、檢影鏡等儀器 |
| 護理師（無驗光證照） | 協助測量、紀錄數據 |

**這是嚴重的邏輯矛盾**：

> ❌ **既然連通過國家高等考試的驗光師都被法律禁止執行 6 歲以下驗光，那完全沒有驗光證照的助理或護理師，憑什麼可以操作？**
> 
> ❌ **既然法律認為這項工作「太危險、太專業」，連驗光師都不能碰，怎麼能交給專業程度更低的人？**
> 
> ❌ **這不是雙重標準，這是「國王的新衣」。**

#### 我們的明確立場

**臺灣視光視力保健學會鄭重主張**：

> **一、6 歲以下兒童之屈光檢查，依法屬於「醫療行為」，必須由眼科醫師「親自執行」。**
> 
> 這包括電腦驗光、檢影驗光等所有屈光測量步驟，而非僅在最後階段「判讀報表」。

> **二、嚴禁由非醫師人員代為操作測量儀器。**
> 
> 任何未取得醫師執照或驗光師（生）執照之人員，包括護理師、診所助理，均不應操作眼球屈光測量儀器。

> **三、呼籲衛生主管機關落實稽查。**
> 
> 若眼科診所人力不足，無法由醫師親自執行 6 歲以下驗光，應誠實告知家長「本院目前無法提供此項服務」，而非交由無照人員充數。

#### 致家長的提醒

> 如果您帶 6 歲以下的孩子去眼科檢查，請注意：
> 
> - **親自確認**：幫孩子操作驗光機的人是不是「醫師本人」？
> - **勇於詢問**：「請問幫我孩子檢查的是誰？有什麼資格？」
> - **這是您的權利**：法律說 6 歲以下要醫師看，那就請醫師「真的看」。

---

#### 具體建議：落實「誰測的誰簽名」原則

**1. 人員資格透明化**

| 建議事項 | 說明 |
|---------|------|
| 稽查人員資格 | 衛生主管機關應嚴格查核眼科診所內執行兒童視力檢查的人員資格 |
| 公示執行人員 | 診所應公示執行驗光人員的姓名與證照字號 |
| 檢查紀錄簽章 | 視力檢查報告應由實際執行者簽章負責 |

**2. 落實專業分工**

| 人員 | 可執行項目 |
|------|-----------|
| **眼科醫師** | 可親自執行所有檢查（包含 6 歲以下） |
| **合格驗光師（在醫師指導下）** | 可執行 6-15 歲驗光、協助 6 歲以下檢查 |
| **無驗光證照人員** | **絕對不可操作驗光儀器** |

**3. 制度性解決方案**

| 短期 | 中長期 |
|------|-------|
| 加強稽查與裁罰 | 修法明確規範 6 歲以下由醫師親自執行 |
| 要求診所公示人員資格 | 建立「眼科—驗光」聯合照護模式 |
| 推動檢查紀錄實名制 | 增加眼科住院醫師培訓名額 |
| 訂定罰則 | 無照執行驗光視同違法醫療行為 |

> **6 歲以下最嬌嫩，法規規定最嚴格，現場卻是最隨便的人在測？**
>
> **既然法律說這麼危險、這麼專業，那就請眼科醫師『親自』下來量！**
>
> **不要只坐在診間看報表。為了孩子的眼睛，請醫師親手操作——這才是真正的『醫療品質』！**

## 5.3 健康教育

### 5.3.1 學生衛教

#### 依年齡設計內容

| 學齡階段 | 重點內容 | 教學方式 |
|---------|---------|---------|
| 國小低年級 | 戶外活動好處、用眼休息 | 遊戲、故事、影片 |
| 國小中高年級 | 近視原因、用眼習慣 | 互動討論、體驗活動 |
| 國中 | 高度近視危害、控制方法 | 專題講座、案例分享 |
| 高中 | 終生視力健康、自我管理 | 自主學習、專題報告 |

#### 核心訊息

1. **戶外活動**：每天至少 2 小時
2. **用眼習慣**：30 分鐘休息 10 分鐘
3. **適當距離**：看書 35-40 公分
4. **定期檢查**：每學期檢查視力
5. **近視危害**：高度近視可能導致失明

### 5.3.2 家長衛教

#### 衛教重點

| 主題 | 內容 |
|------|------|
| 近視嚴重性 | 高度近視的併發症風險 |
| 預防方法 | 戶外活動的重要性 |
| 控制方法 | 介紹各種控制選項 |
| 迷思破解 | 配眼鏡不會讓近視加深 |
| 就醫時機 | 何時該帶孩子檢查 |

#### 執行方式

| 方式 | 說明 |
|------|------|
| 衛教單張 | 隨通知單發放 |
| 親師座談 | 納入視力保健主題 |
| 學校網站 | 建立視力保健專區 |
| LINE 群組 | 定期推播衛教訊息 |

## 5.4 環境改善

### 5.4.1 教室照明標準

| 項目 | 標準值 | 檢測方法 |
|------|-------|---------|
| 桌面照度 | **≥ 500 lux** | 照度計 |
| 黑板照度 | **≥ 500 lux** | 照度計 |
| 均勻度 | ≥ 0.7 | 最低/平均照度比 |
| 眩光指數 | ≤ 19 | 專業評估 |

### 5.4.2 照明改善建議

| 問題 | 改善方式 |
|------|---------|
| 照度不足 | 增加燈具數量或更換高效能燈具 |
| 不均勻 | 調整燈具位置與角度 |
| 眩光 | 加裝燈罩、調整座位 |
| 黑板反光 | 使用霧面黑板、調整燈光角度 |

### 5.4.3 課桌椅調整

| 項目 | 標準 |
|------|------|
| 椅高 | 小腿垂直時，腳底平放地面 |
| 桌高 | 手肘彎曲 90° 時，前臂平放桌面 |
| 桌椅差 | 依學生身高調整 |

### 5.4.4 戶外時間安排

| 時段 | 建議做法 |
|------|---------|
| 課間休息 | 鼓勵到戶外活動，淨空教室 |
| 體育課 | 儘量安排在戶外 |
| 早自習前 | 開放校園讓學生戶外活動 |
| 午休後 | 安排戶外散步或活動時間 |
| 課後 | 開放校園供學生使用 |

### 5.4.5 環境檢核表

#### 每學期檢核項目

- [ ] 教室照明是否符合標準（≥ 500 lux）？
- [ ] 黑板是否有反光或眩光？
- [ ] 課桌椅是否依學生身高調整？
- [ ] 學生座位是否定期輪換？
- [ ] 是否有充足的戶外活動時間？
- [ ] 下課時間學生是否到戶外活動？

## 5.5 建議增設：驗光師入校服務

### 5.5.1 服務模式

| 模式 | 說明 |
|------|------|
| 定期到校服務 | 驗光師定期至學校提供篩檢與諮詢 |
| 專業支援 | 協助護理師執行視力篩檢 |
| 衛教講座 | 對學生、家長進行專業衛教 |
| 轉介銜接 | 與眼科建立轉介合作機制 |

### 5.5.2 法規依據

根據**驗光人員法第 12 條**：
- 驗光師可執行非侵入性眼球屈光狀態測量
- **15 歲以下需在眼科醫師指導下執行**
- **6 歲以下不得執行驗光**

**建議**：可透過與眼科醫師合作（如書面醫囑、遠距指導）的方式，讓驗光師合法提供學校服務。

### 5.5.3 預期效益

1. 提升視力篩檢的專業度與準確性
2. 早期發現屈光異常
3. 提供更精準的轉介建議
4. 增加視力保健的專業人力
5. 促進跨專業合作

## 5.6 視光臨床實務建議（專家觀點）

本節針對近視控制的臨床實務，提出視光專業的核心觀點與建議。這些觀點基於臨床經驗與專業知識，旨在提升近視管理的品質。

### 5.6.1 驗光不等於電腦打單

**核心觀點**：完整的驗光檢查絕非僅靠電腦驗光機（Auto-Refractor）打一張單據就結束。

**完整驗光應包含**：

| 檢查項目 | 目的 |
|---------|------|
| 電腦驗光 | 初步客觀屈光參考值 |
| 檢影驗光 / 綜合驗光 | 精確測量屈光度數 |
| **調節力評估** | 測量調節幅度、調節靈活度、調節滯後 |
| **雙眼協調評估** | 測量眼位、融像範圍、立體視 |
| 主觀驗光 | 確認最佳矯正度數與舒適度 |

**為什麼重要**：
- 調節功能異常可能是近視進展的推手
- 雙眼視覺問題可能導致閱讀困難、眼睛疲勞
- 僅依賴電腦驗光機的「便宜行事」會遺漏重要問題

> **專業堅持**：視光專業的價值在於完整評估，而非快速出單。

---

### 5.6.2 拒絕濫用等價球面

**核心觀點**：散光必須精準矯正，不可過度依賴等價球面（Spherical Equivalent）換算。

**等價球面的問題**：

等價球面公式：`SE = 球面度數 + (散光度數 ÷ 2)`

| 情況 | 風險 |
|------|------|
| 將散光換算成球面 | 視網膜成像仍模糊 |
| 未矯正的散光 | 可能刺激眼軸增長 |
| 長期模糊視覺 | 影響視覺發育與學習 |

**正確做法**：
- 散光 ≥ 0.75D 應精準矯正
- 不可為了「簡化處方」而犧牲矯正品質
- 高散光兒童更需謹慎處理

> **專業堅持**：精準矯正是視光師的基本責任，等價球面僅供參考計算，不可取代完整矯正。

---

### 5.6.3 角膜塑型需定期停戴（Washout）

**核心觀點**：配戴角膜塑型片（Ortho-K）的學童，每年應安排停戴期，讓角膜回彈後進行完整視力評估。

**為什麼需要 Washout**：
- 角膜塑型會暫時改變角膜弧度
- 配戴期間無法測得真實屈光度數
- 僅測量眼軸長無法反映完整視覺狀況

**建議做法**：

| 項目 | 建議 |
|------|------|
| 停戴時機 | 寒假或暑假 |
| 停戴時間 | 至少 **2-4 週**（依個人恢復速度） |
| 停戴期間 | 配戴備用眼鏡 |
| 完整檢查 | 角膜地形圖、屈光度數、眼軸長、雙眼視覺 |

**檢查項目**：
- 角膜是否完全回彈（角膜地形圖確認）
- 真實屈光度數變化
- 眼軸長度變化
- 角膜健康狀況

> **專業堅持**：僅追蹤眼軸長是不夠的，Washout 後的完整評估才能掌握全貌。

---

### 5.6.4 階梯式處置原則

**核心觀點**：近視控制應採「先光學、後藥物」的階梯式策略，優先採用副作用最小的方式。

**處置階梯**：

```
第一階：行為與環境介入（基礎）
├── 增加戶外活動時間
├── 改善用眼習慣
└── 優化視覺環境

第二階：光學介入（優先）
├── 周邊離焦眼鏡
├── 角膜塑型
└── 多焦軟性隱形眼鏡

第三階：藥物介入（升級）
├── 0.05% 阿托品（首選）
└── 合併光學介入

第四階：合併治療（強化）
└── 藥物 + 光學多重介入
```

**動態調整原則**：
- 每 6 個月評估控制成效
- 進展過快（> 50 度/半年）則升級處置
- 控制穩定可維持現有方案
- 長期穩定可考慮降階

**優先順序的理由**：

| 介入類型 | 副作用 | 可逆性 | 優先順序 |
|---------|-------|--------|---------|
| 行為環境 | 無 | - | ⭐⭐⭐⭐⭐ |
| 光學介入 | 低 | 高 | ⭐⭐⭐⭐ |
| 低濃度藥物 | 低 | 高 | ⭐⭐⭐ |
| 高濃度藥物 | 高 | 中 | ⭐⭐ |

> **專業堅持**：不是每個孩子都需要點藥，階梯式處置讓介入更精準、副作用更小。
