# Optometry Notes

驗光知識庫 - Obsidian Vault

## 結構

```
00-MOC/          # Map of Content（索引頁）
01-文獻/          # 論文全文與摘要
02-診斷標準/      # 各種診斷的標準卡片
03-症狀/          # 症狀卡片
04-檢查項目/      # 檢查方法與工具
05-部落格素材/    # 寫作素材
templates/       # 模板
```

## 使用方式

1. Clone 這個 repo 到本機
2. 用 Obsidian 開啟這個資料夾作為 vault
3. 從 `00-MOC/雙眼視覺總覽.md` 開始瀏覽

## 功能

- 雙向連結 `[[]]`：點擊可跳轉
- 標籤系統 `#tag`：便於篩選
- Graph View：視覺化知識連結
- Frontmatter：YAML metadata

## 持續更新

這個 vault 會持續更新，`git pull` 即可同步最新內容。
