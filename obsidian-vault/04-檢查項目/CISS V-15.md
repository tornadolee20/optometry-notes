# CISS V-15 問卷

> Convergence Insufficiency Symptom Survey（集合不足症狀問卷）

---

## 基本資料

| 項目 | 內容 |
|------|------|
| 對象 | [[CI 診斷標準|CI]] 篩檢 |
| 適用 | 兒童與成人皆可 |
| 題數 | 15 題 |
| 計分 | 0-4 分 |
| 總分 | 0-60 分 |
| 判斷 | **≥16 分 = 可能有 CI** |

---

## 計分方式

| 選項 | 分數 |
|------|------|
| 從不 | 0 |
| 很少 | 1 |
| 有時候 | 2 |
| 經常 | 3 |
| 總是 | 4 |

---

## 15 題問卷（中文版）

> 以下問題詢問「閱讀或近距離工作時」的情況

| # | 問題 |
|---|------|
| 1 | 眼睛是否感到**疲勞**？ |
| 2 | 眼睛是否感到**不舒服**？ |
| 3 | 是否會**頭痛**？ |
| 4 | 是否會**想睡覺**？ |
| 5 | 是否難以**集中注意力**？ |
| 6 | 是否難以**記住讀過的內容**？ |
| 7 | 是否會看到**兩個影像（複視）**？ |
| 8 | 文字是否會**移動、跳動或漂浮**？ |
| 9 | 是否覺得自己**閱讀速度很慢**？ |
| 10 | 眼睛是否會**痛**？ |
| 11 | 眼睛是否會**痠**？ |
| 12 | 眼睛周圍是否有**拉扯感**？ |
| 13 | 文字是否會**模糊或焦距不穩**？ |
| 14 | 閱讀時是否會**跳行或看錯行**？ |
| 15 | 是否需要**重複閱讀同一行**？ |

---

## 怎麼判讀？

| 分數 | 意義 |
|------|------|
| 0-15 | ✅ 正常 |
| **16-20** | ⚠️ 可能有 CI，建議進一步檢查 |
| **≥21** | ❌ 很可能有 CI（CITT 研究的納入標準） |

---

## 臨床怎麼用？

### 1. 篩檢
- 懷疑 CI 時，先讓患者填問卷
- ≥16 分 → 安排詳細檢查

### 2. 追蹤治療效果
- 治療前後各填一次
- 分數下降 = 症狀改善

### 3. 研究標準
- CITT 研究用 ≥21 分作為納入條件

---

## 注意事項

⚠️ **CISS 只適用於 CI 篩檢**
- 其他雙眼視覺異常沒有專用問卷
- 問卷高分不代表一定是 CI
- 要結合客觀檢查（[[NPC]]、斜位、融像）才能診斷
