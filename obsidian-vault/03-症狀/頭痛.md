# 頭痛

#症狀

---

## 與雙眼視覺異常的關聯

頭痛是雙眼視覺異常最常見的症狀，在 53/56 篇研究中被提及。

---

## 特徵

| 特徵 | 描述 |
|------|------|
| **位置** | 額頭、太陽穴、眼眶周圍 |
| **時機** | 近距離工作後加重 |
| **持續** | 工作時或工作後 |
| **緩解** | 休息、閉眼後改善 |
