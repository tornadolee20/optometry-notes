### 🏥 全身健康狀況 (Systemic Health Profile)

全身性疾病會直接影響眼睛健康，此區塊為**多選勾選 (Checkbox)**，所有族群皆適用：

| 欄位名稱 | 欄位代號 | 型別 | 說明 | 對眼睛的影響 |
| :--- | :--- | :--- | :--- | :--- |
| 高血壓 | `has_hypertension` | Bool | 勾選 | 🩸 高血壓性視網膜病變風險 |
| 高血糖 | `has_hyperglycemia` | Bool | 勾選 | 🍬 糖尿病前期，影響水晶體代謝 |
| 高血脂 | `has_hyperlipidemia` | Bool | 勾選 | 🫀 視網膜血管硬化風險 |
| 糖尿病 | `has_diabetes` | Bool | 勾選 | ⚠️ **糖尿病視網膜病變 (DR)** — 致盲主因之一 |
| 糖尿病類型 | `diabetes_type` | Enum | Type 1 / Type 2 / 妊娠性 | (若勾選糖尿病才展開) |

#### 🚨 健康狀況連動 AI 警報：
*   糖尿病 = True ➔ 🔴 每次報告必加警語：「建議每年至少做一次散瞳眼底檢查，排除糖尿病視網膜病變！」

*   高血壓 + 年齡 > 50 ➔ 🟡 自動推播：「建議安排眼底攝影，追蹤視網膜血管健康。」

## 🍼 貳、 學齡前幼兒 (0-6歲) — 視覺發育追蹤欄位

> [!NOTE]
> **法規定位**：依台灣《驗光人員法》，6 歲以下幼兒需由眼科醫師執行驗光。本系統在此年齡層的定位為：
> 1. **家長自行紀錄**眼科回診的檢查結果（如遠視儲備、弱視篩查）。
> 2. **衛教提醒引擎**：驗光師雖不能驗光，但可以透過系統提供護眼衛教與回診提醒。

**核心關注痛點**：家長最需要的一句話 ➔ **「我的孩子眼睛發育正不正常？有沒有弱視？」**
**早期介入黃金期**：弱視治療的最佳窗口為 **3-6 歲**，超過 8 歲後治療效果急遽下降。
**商業導購目標**：兒童護眼保健品、抗藍光兒童眼鏡、眼科轉介 (建立醫視融合通路)

| 欄位名稱 | 欄位代號 | 型別 | 說明 | 家長看到什麼？ |
| :--- | :--- | :--- | :--- | :--- |
| 出生日期 | `birth_date` | Date | 精確計算月齡，對照發育里程碑 | 🎂 「寶寶：3歲2個月」 |
| 紀錄來源 | `record_source` | Enum | 眼科回診 / 幼兒園篩檢 / 健兒門診 | 🏥 「資料來源：眼科回診」 |
| **右眼屈光狀態** | `od_refraction_status` | Enum | 正常遠視儲備 / 遠視不足 / 已近視 / 散光 | 👁️ 「右眼：遠視儲備充足 ✅」 |
| **左眼屈光狀態** | `os_refraction_status` | Enum | 同上 | 同上 |

### 📊 幼兒視覺發育里程碑 (Age-Based Reference)

系統將根據幼兒月齡，自動對照以下發育里程碑，並在 AI 建議中標示是否達標：

| 年齡 | 正常遠視儲備參考 | 視力發育參考 | 關鍵篩查項目 |
| :--- | :--- | :--- | :--- |
| 0-1 歲 | +3.00D ~ +4.00D | 追視、瞳孔反射 | 紅反射 (排除先天白內障) |
| 1-2 歲 | +2.50D ~ +3.50D | 伸手抓物準確度 | 斜視初篩 |
| 3-4 歲 | +1.50D ~ +2.50D | 0.5-0.6 | ⭐ **第一次完整眼科檢查** (弱視黃金篩檢期) |
| 4-5 歲 | +1.00D ~ +2.00D | 0.7-0.8 | 弱視追蹤、遠視儲備評估 |
| 5-6 歲 | +0.75D ~ +1.50D | 0.8-1.0 | 入學前完整視力評估 |

### 🚨 學齡前專屬 AI 警報觸發條件：
*   弱視篩查 = 疑似或確診 ➔ 🔴 **緊急**：「弱視治療黃金期為 3-6 歲，請務必遵從醫囑積極治療！」
*   遠視儲備 < 同齡下限 ➔ 🟡 **預警**：「寶寶的遠視儲備偏低，未來近視風險較高，建議增加戶外活動。」
*   遠視儲備消耗過快 (半年下降 > 0.50D) ➔ 🔴 **警報**：「遠視儲備消耗速度過快！」
*   距上次眼科回診超過 6 個月 ➔ 🟡 **提醒**：「寶寶距離上次眼科檢查已超過半年，建議預約回診追蹤。」
*   遮眼訓練未達標 ➔ 🟡 **鼓勵**：「今天遮眼時數還差 1.5 小時喔，加油！堅持就是保護寶寶視力的最大力量！」

### 📐 AXL 智慧估算引擎 (三階段分層演算法)

系統內建三種國際認可的眼軸估算公式，根據「可輸入的資料豐富度」自動選擇最精確的演算法：

| 優先度 | 公式名稱 | 所需輸入 | 精度 | 適用場景 |
| :--- | :--- | :--- | :--- | :--- |
| ⭐⭐⭐ | **實測值優先** | IOLMaster / Lenstar 實測 | 最精確 | 有光學生物測量儀的高階診所 |
| ⭐⭐ | **Lingham 2024 兒童迴歸** | K值 + SE + 身高 | R²=82.4% | 7-14 歲兒童，有角膜弧度儀 |
| ⭐ | **Morgan 2020 通用公式** | CR (角膜曲率半徑) + SE | ±0.73mm | 通用，有角膜弧度儀即可 |
| 🚩 | **AL/CR 快速風險判定** | AXL (實測或估算) / CR | 比值 >3 = 高風險 | 大量篩查、戰情室熱力圖 |

#### 公式定義：

**① Lingham 2024 (兒童專用，首選)**
```
AXL = 4.203 + (2.330 × Km) - (0.420 × SE) + (0.008 × Height_cm)
```
*   `Km` = 平均角膜屈光力 (Diopters)
*   `SE` = 球面等效度數 = SPH + (CYL / 2)
*   `Height_cm` = 兒童身高 (cm)
*   出處：MDPI 2024, myopic children aged 7-14

**② Morgan 2020 (通用估算)**
```
AXL = f(CR, SE)  // 使用角膜曲率半徑與球面等效度
```
*   `CR` = 平均前角膜曲率半徑 (mm) = 337.5 / Km
*   `SE` = 球面等效度數
*   精度：±0.73mm (與 IOLMaster 對照)
*   出處：Morgan et al. 2020, Ulster University

**③ AL/CR 近視風險比值**
```
AL_CR_Ratio = AXL / CR
IF AL_CR_Ratio > 3.0 → 高近視風險
```
*   無需知道精確 AXL，可搭配上述任一公式的估算值
*   出處：多篇中國大規模兒童篩查研究 (4-18歲 age/gender percentile charts)

### 🚨 兒童專屬 AI 警報觸發條件：
*   半年度數增加 ≥ 50度 ➔ 🔴 **紅色警報**：「建議立即回診評估更強力的近視控制方案！」
*   半年度數增加 25-49度 ➔ 🟡 **黃色留意**：「防控中，但請留意下次追蹤。」
*   半年度數增加 < 25度 ➔ 🟢 **綠色安心**：「太棒了！目前的方案控制得非常好！」
*   AL/CR > 3.0 ➔ 🔴 **風險警報**：「近視風險指數偏高，建議儘速評估近視控制方案！」
*   眼軸半年增加 > 0.3mm ➔ 🔴 **眼軸警報**：「眼球發育速度高於安全範圍！」
