# 📋 給 Stitch AI 的 Prompt（複製貼上即可）

---

請幫我設計一個名為「Vision Passport 視力存摺」的 Mobile-First Web App UI。

## 產品定位
這是一個獨立的「全家人的數位視光存摺」手機 APP，讓家長追蹤全家人（從嬰兒到長輩）的視力變化趨勢。這是一個獨立 APP（PWA 或 React Native），不是 LINE 內的微程式。

## 設計風格
- 暗色系底色 (Dark Mode)，搭配霓虹藍 (#00D2FF) 與紫色 (#B567FF) 漸層
- Glassmorphism 玻璃擬物化卡片風格
- 字體使用 Noto Sans TC + Outfit
- 所有轉場需像原生 App 一樣平滑滑動
- 介面語言為繁體中文

## 頁面一：家庭成員選擇頁（首頁）
- 頂部 Logo 文字「VISION PASSPORT」帶有漸層色發光效果
- 副標題「您的數位視光存摺」
- 下方為家庭成員卡片清單，每張卡片包含：
  - 左側：圓角方形的 emoji 頭像（有漸層底色）
  - 中間：姓名 + 狀態標籤（如「近視控制警戒🔴」、「視力健康🟢」、「多焦適應中🟣」）
  - 右側：箭頭 chevron
- 底部：「+ 新增家庭成員」虛線按鈕

## 頁面二：個人視力護照頁（點擊家庭成員後進入）
- 頂部導航列：返回按鈕 + 「OOO 的視力存摺」+ 設定齒輪
- 區塊一「核心數據卡」(glass card)：
  - 左半：最新度數（大字體，紅色代表警戒）
  - 右半：眼軸長度（黃色代表留意）
  - 各自下方有趨勢箭頭（如「↑ 增加 50度 / 半年」）
- 區塊二「歷年度數折線圖」(glass card)：
  - 預留 Chart.js 圖表區塊
  - 標題帶有圖表 emoji
- 區塊三「AI 臨床分析報告」(glass card)：
  - 標頭有🤖 icon + 標題 + 「最新」badge
  - 內容為白話文分析段落
  - 底部有「行動建議」區塊，帶有閃電 emoji 的清單項目
  - 最下方一個全寬的漸層 CTA 按鈕「立即預約驗光所」
- 區塊四「歷史驗光紀錄」：
  - 時間軸清單，每筆有日期 + 門市名稱 + 「看報告」小按鈕

## 頁面三：新增紀錄頁（漸進式揭露設計）
- 設計原則：**Lv.1 只顯示最少欄位**
- 預設只顯示：日期選擇 + 右眼度數 + 左眼度數（三個欄位）
- 下方有一個可展開的摺疊區塊「想填寫更多？」，點擊後展開：
  - 散光度數、眼軸長、近視控制方案、角膜弧度
- 最底部再有一個「進階健康資訊」摺疊區塊：
  - 多選勾選框：高血壓、高血糖、高血脂、糖尿病、類固醇
- 頁面底部有「📷 拍照辨識處方箋」按鈕（大、醒目）

## 重要 UI 規則
1. 一切以手機螢幕 (375px~414px) 為設計基準
2. 卡片間距舒適，不擁擠
3. 所有互動元素都要夠大（最小 44px），方便手指點擊
4. 頁面之間的轉場使用左右滑動動畫
5. 底部不需要 Tab Bar，用頂部返回箭頭導航即可
